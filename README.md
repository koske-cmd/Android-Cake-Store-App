# 🍰 Android Cake Store

Android application for browsing and ordering cakes with dynamic data loading from a JSON API.

![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)
![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

## 📱 Features

- Browse a list of delicious cakes
- Search cakes by name or pictures
- View detailed information about each cake
- Smooth scrolling with RecyclerView
- Modern Material Design UI
- Dynamic JSON data loading

## 🛠️ Tech Stack

- **Language:** Java
- **UI:** Material Design, RecyclerView
- **Networking:** JSON parsing
- **Image Loading:** Glide
- **Architecture:** MVC Pattern

## 🚀 Getting Started

### Prerequisites

- Android Studio Arctic Fox or later
- Android SDK 21+
- JDK 11+

### Installation

1. Clone the repository
```bash
git clone https://github.com/koske-cmd/android-cake-store-app.git
```
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the app on an emulator or physical device
